# 代理hyper虚拟机端口 

``` bat
# 添加转发规则
netsh interface portproxy add v4tov4 listenaddress=127.0.0.1 listenport=2222 connectaddress=172.17.20.10 connectport=22

# 查看映射
netsh interface portproxy show all

# 删除映射
netsh interface portproxy delete v4tov4 listenaddress=127.0.0.1 listenport=2222
```

# 计算文件md5值

``` bat

```

# 使用_curl_查看域名证书

``` Bash
curl --insecure -v https://域名 2>&1 | awk 'BEGIN { cert=0 } /^\* SSL connection/ { cert=1 } /^\*/ { if (cert) print }'
```
